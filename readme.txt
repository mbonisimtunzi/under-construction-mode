== Description ==

A simple plugin that displays an "Under Construction" page to non-logged-in users.
You can customize the background color, image, and overlay opacity.

== Installation ==

1. Upload the `under-construction-mode` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' > 'Under Construction' to configure the plugin.
